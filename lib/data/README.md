# Data Files

This directory contains all JSON data files for Penn State academic programs.

## Files

- `penn_state_courses.json` - Complete course catalog (9,275+ courses)
- `penn_state_majors.json` - Major program requirements (40 majors)
- `penn_state_minors.json` - Minor program requirements
- `gen_ed_requirements.json` - General Education requirements (shared across all majors)

## Gen Ed Attributes

All courses include Gen Ed attributes:
- **GWS**: Writing/Speaking (31 courses)
- **GQ**: Quantification (10 courses)
- **GHW**: Health & Wellness (7 courses)
- **GN**: Natural Sciences (50 courses)
- **GA**: Arts (111 courses)
- **GH**: Humanities (55 courses)
- **GS**: Social & Behavioral Sciences (55 courses)
- **interdomain**: Integrative Studies (24 courses)
